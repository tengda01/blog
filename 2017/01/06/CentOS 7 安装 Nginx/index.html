CentOS 7.3安装 Nginx

为什么选择Nginx?

Nginx 是一个很牛的高性能Web和反向代理服务器, 它具有很多非常优越的特性:

  • 在高连接并发的情况下，Nginx是Apache服务器不错的替代品
  • Nginx作为负载均衡服务器
  • 作为邮件代理服务器
  • Nginx 是一个安装非常的简单 , 配置文件非常简洁还能够支持perl语法,Bugs非常少的服务器.

测试环境

  • windows 10 专业版 1607
  • VirtualBox 5.1.12 r112440 (Qt5.6.2)
  • CentOS release 7.3.1611
  • Nginx 1.11.8
